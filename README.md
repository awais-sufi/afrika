## About

ForAfrika works across multiple African countries including Mozambique, Kenya, Ethiopia, South Sudan, and other nations where humanitarian support is needed. Our mission-driven website showcases our programs, stories, and ways to get involved.

## Getting Started

### Prerequisites

- Node.js 18.x or higher
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/your-repo/forafrika.git
cd forafrika
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
# or
yarn build
```

### Start Production Server

```bash
npm start
# or
yarn start
```
